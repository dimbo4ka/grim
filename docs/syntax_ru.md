1. **Объявление переменных**:
```
x := 10; // type deduction

var x: int = 10;
```
2. **Встроенные типы**

| имя типа | описание                            | значимый/ссылочный |
| -------- | ----------------------------------- | ------------------ |
| int      | 128-битный целочисленный тип данных | значимый           |
| bool     | 1-байтный логический тип данных     | значимый           |
| str      | иммутабельная строковый тип данных  | значимый           |
| vector   | динамический массив                 | ссылочный          |

3. **Арифметические операторы между int**
Унарные: 
	Префиксные: `-`, `+`, `~`
	Постфиксные: `--`, `++`. 
Бинарные: `&`, `|`, `^`, `*`, `/`, `+`, `-`, `=`, `*=`, `/=`, `+=`, `-=`
	Операторы сравнения:
		`>`, `<`, `==`, `!=`, `>=`, `<=`
Тернарный оператор: `: ?`
3. **Булевые операторы**
`&&`, `||`, `!`

4. **Операции над строками**:
Операторы сравнения:
	`>`, `<`, `==`, `!=`, `>=`, `<=`
Операторы конкатенации: 
	`+=`, `+`
5. **Определение функций**
```

```


7. **Циклы**

Бесконечный цикл:
```
loop {
	// ...
}
```

Цикл **while**:
```
while /*condition*/ {
	// ...
}
```

Цикл **for**
```
for /*init statement*/; /*condition*/; /*expression*/ {
	// ...
}
```

Цикл **foreach**
```
for i, val in iterable_object {
	// ...
}
```

Операторы перехода:
`break`, `while`

8. **Условный оператор**

```
if /*condition1*/{
	// ...

	// ...

	// ...
}
```

9. **Константы**

```
const int num = 10;
const str s = "abacaba";
```
Константы в **grim** являются алиасами для литералов

9. **Statements**
Каждый statement должен заканчиваться символом `;`

10. **Работа с vector**:
```
// Создание вектора
v := int[]{1, 2, 3, 4};
v := new(int[]); // null-value
v := new(int[], len, capacity);

v.push(10);
v.pop();

v.size();
v.capacity();
```

10. **Модули**:

**Модуль** — набор файлов с расширением `.gr`, находящихся в одной директории, в каждом из которых в начале указано:
```
module module_name;
```
***Код внутри одного модуля образует единое пространство имен.***

Функции, структуры и их поля должны быть доступны извне только в том случае, если перед их определением указан модификатор `pub`.
